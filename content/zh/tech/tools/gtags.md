## 下载页

https://www.gnu.org/software/global/download.html

## 安装

```bash
# 下载源码
wget https://ftp.gnu.org/pub/gnu/global/global-6.6.4.tar.gz
# 解压
tar xvf global-6.6.4.tar.gz
cd global-6.6.4
# 编译安装
./configure
make
sudo make install
```
